About

This is a web application framing the trained model for detection of distracted image trained on VGG-16 architecture model and then load in server for prediction so that it may be used by any user remotely

Insight

The model is builed using thousands of driver images passed in a CNN network and finally it is showing around 96% accuracy based on 7 classes of prediction which can be shown after any user submit any image for detection.
